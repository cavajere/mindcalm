# Regole UI

## Tabelle admin

- Nelle tabelle del portale di amministrazione, la colonna `Azioni` deve usare solo pulsanti con icone. Non usare testo visibile come `Modifica`, `Elimina`, `Pubblica` o simili dentro la cella.
- I pulsanti icona nella colonna `Azioni` devono avere `title` e `aria-label` coerenti, cosi' il significato resta chiaro anche senza etichetta testuale.
- L'intestazione `Azioni` e il contenuto della relativa colonna devono essere sempre centrati orizzontalmente.
- Per mantenere consistenza visiva, nelle viste admin usare le classi condivise `table-actions-header`, `table-actions-cell`, `table-actions-group` e `icon-action-button`.
